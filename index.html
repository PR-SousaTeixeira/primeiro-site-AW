Anotações da aula

HTML (HyperText Markup Language)

HTML Serve para "marcar" o conteúdo da página é como dizer ao browser: isto é um link, isto é uma imagem, isto é um título através de Tags que funcionam como etiquetas de formatação posteriormente lidas pelo browser.

A declaração define que este documento é um documento HTML5

O elemento <"HTML"> é o elemento raiz de uma página HTML

O elemento <"head"> contém meta informações sobre a página HTML

O elemento <"title"> especifica um título para a página HTML (que é mostrado na barra de título do navegador ou na guia da página)

O elemento <"body"> define o corpo do documento e é um conteiner para todos os conteúdos visíveis, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.

O elemento <"h1"> define um título grande, É definido do h1 ao h6 por nível de importância

O elemento <"p"> define um parágrafo, já que em HTML os espaços ou enters são ignorados

Os elementos <"i">, <"b">, <"u">, são alguns exemplos de formatação de texto como Itálico, Bold e Sublinhado respetivamnete

O elemento <"ul"> ou <"ol"> definem listas sem ordem ou enumeradas respetivamente, juntamente com a tag <"li"> para enquadrar cada exemplo, como por exemplo

  • Exemplo sem ordem 2

O elemento <"img"> é usado para incorporar uma imagem em uma página da web, as imagens não são tecnicamente inseridas em uma página da web as imagens estão vinculadas a páginas da web. A tag <"img"> cria um espaço de retenção para a imagem referenciada

A Tag <"img"> não têm de ser fechada, só necessita de atributos como "src" e "alt"para especificar o caminho da iamgem ao browser e especificar texto referente a imagem respetivamente

TAGS

Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final

<"tagname"> O conteúdo vai aqui <"/tagname">

O elemento HTML é tudo, desde a tag inicial até a tag final

Liks

Liks são elementos de Hyperliagação para outros elementos, possibilita saltitar de um documento para outro

São definidos com a tag <"a"> seguido inicialmente por o atributo "href=" que indica o caminho do link, finalizado pelo "titulo" do link que será a parte visível

como por exemplo "href="http://www.google.com">Google" que juntamente com a tag corresponde a Google

Conceitos básicos

Java Script-É uma linguagem de programação que permite programar o comportamento e a interação com uma página. Com JavaScript conseguimos manipular a página, adicionar eventos, manipular dados, etc.
