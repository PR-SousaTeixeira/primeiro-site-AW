HTML (HyperText Markup Language)

HTML Serve para "marcar" o conteúdo da página é como dizer ao browser: isto é um link, isto é uma imagem, isto é um título através de Tags que funcionam como etiquetas de formatação posteriormente lidas pelo browser.

A declaração define que este documento é um documento HTML5

O elemento <"HTML"> é o elemento raiz de uma página HTML

O elemento <"head"> contém meta informações sobre a página HTML

O elemento <"title"> especifica um título para a página HTML (que é mostrado na barra de título do navegador ou na guia da página)

O elemento <"body"> define o corpo do documento e é um conteiner para todos os conteúdos visíveis, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.

O elemento <"h1"> define um título grande, É definido do h1 ao h6 por nível de importância

O elemento <"p"> define um parágrafo, já que em HTML os espaços ou enters são ignorados

Os elementos <"i">, <"b">, <"u">, são alguns exemplos de formatação de texto como Itálico, Bold e Sublinhado respetivamnete

O elemento <"ul"> ou <"ol"> definem listas sem ordem ou enumeradas respetivamente, juntamente com a tag <"li"> para enquadrar cada exemplo, como por exemplo

  • Exemplo sem ordem 2

CSS

É o que define a página "visualmente", pode ser usado para definir cores, layout e estilos

A sintaxe de uma declaração CSS resulta em algo como:

<"tag"> texto <"/tag"> (Aquela que vai ser alterada)

seletor de tag{

propriedade: Valor;(Aquilo que vai ser alterado)

}

É importante a utilização de selectores para identificar o elemento ao qual deve ser atribuido um estilo

Alguns dos seletores mais comuns são:

  • Nome do elemento-seleciona elementos HTML com base no nome do elemento.
  • ID(#Id)- usa o atributo id de um elemento HTML para selecionar um elemento específico.
  • Classe(.class)-seleciona elementos HTML com um atributo de classe específico.
  • Atributo([Atributo])-leciona elementos HTML com base no atributo
  • tudo(*)-seleciona todos os elementos HTML na página.

As formas de inserir CSS podem ser categorizadas em três formas:

  • Inline CSS- Um estilo Inline pode ser usado para aplicar um estilo exclusivo a um único elemento, é situado na linha do proprio elemento.
  • Internal CSS-Pode ser usado para dar um único estilo a toda a página HTML, é situado dentro da seção da <"Head">
  • External CSS-Pode ser usado para atribuir um estilo que se enconta numa sheet externa a página, é situado na secção <"Head"> e é defenido com o elemento <"link"> para referenciar a sheet externa

Cascading-um estilo Inline tem prioridade sobre um estilo Internal e External

Comentarios-É possivel adicionar comentarios a uma linha de codigo que não é lida pelo browser posteriormente O comentário CSS é colocado dentro do elemento <"style"> e começa com /"*e termina com *"/:(sem as aspas)
